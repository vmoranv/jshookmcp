# AntiDebug

域名：`antidebug`

反反调试域，集中提供检测与绕过浏览器端反调试脚本的工具。

## Profile

- full

## 典型场景

- 调试器绕过
- 计时检测缓解
- 控制台/devtools 探测对抗

## 常见组合

- browser + antidebug + debugger

## 代表工具

- `antidebug_bypass` — 绕过指定类型的反调试保护。
- `antidebug_detect_protections` — 检测当前页面的反调试机制并给出绕过建议。
