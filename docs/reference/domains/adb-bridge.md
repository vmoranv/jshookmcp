# ADB Bridge

域名：`adb-bridge`

Android Debug Bridge 集成域，用于设备管理、应用分析和远程调试。

## Profile

- workflow
- full

## 典型场景

- Android 设备管理
- APK 分析
- 远程调试

## 常见组合

- adb-bridge + process
- adb-bridge + network

## 代表工具

- `adb_device_list` — 待补充中文：List Android devices that are currently reachable through ADB.
- `adb_shell` — 待补充中文：Run an ADB shell command on a specific Android device.
- `adb_apk_pull` — 待补充中文：Pull an APK file from an Android device to a local path.
- `adb_apk_analyze` — 待补充中文：Analyze an installed APK — package name, version, permissions, activities, services, receivers.
- `adb_webview_list` — 待补充中文：List debuggable WebView targets via ADB port forwarding to Chrome DevTools. Requires android:debuggable="true".
- `adb_webview_attach` — 待补充中文：Attach to a WebView via ADB port forwarding; returns WebSocket debugger URL for CDP.
