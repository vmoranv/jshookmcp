# Skia Capture

域名：`skia-capture`

Skia 渲染引擎捕获域，用于 UI 渲染分析和可视化。

## Profile

- workflow
- full

## 典型场景

- Skia 场景提取
- 渲染管道分析
- UI 组件识别

## 常见组合

- skia-capture + browser
- skia-capture + canvas

## 代表工具

- `skia_detect_renderer` — 待补充中文：Detect the active Skia renderer backend from the current page context.
- `skia_extract_scene` — 待补充中文：Extract a lightweight Skia scene tree from the selected canvas.
- `skia_correlate_objects` — 待补充中文：Correlate requested Skia node identifiers with the extracted scene tree.
