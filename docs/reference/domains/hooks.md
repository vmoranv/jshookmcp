# Hooks

域名：`hooks`

AI Hook 生成、注入、数据导出，以及内置/自定义 preset 管理。

## Profile

- full

## 典型场景

- 函数调用采集
- 运行时证据留存
- 团队专用 inline preset

## 常见组合

- browser + hooks + debugger

## 代表工具

- `ai_hook` — 管理 AI 钩子（注入/获取数据/列表/清除/切换/导出）。
- `hook_preset` — 安装内置或自定义的 JavaScript Hook 预设模板。
